Detailed Description

This class exists to provide an abstract interface between asset tags and textures. Technically, the ContentManager already does this for XNA, but ContentManager is missing a couple of important functions: modability, and storing the inverse lookup between tag and texture. Additionally, the TextureManager provides an interface to directly load resources from the disk (rather than going through XNAs content system)
