Detailed Description

A GUITween is an animation that causes a GUI component to move smoothly relative to its parent.

Member Enumeration Documentation

§ TweenType

enum DwarfCorp.GUIComponent.GUITween.TweenType
strong

A tween type defines whether the GUI component is getting created by this animation, destroyed, or merely moved.

Enumerator
TweenIn 

When tweening in, the GUI component will be inactive until the tween ends.

TweenOut 

When tweening out, the GUI component will be destroyed after the tween ends.

TweenAnimate 

When animating, the GUI component merely moves during the tween.
