Create texture coordinate for a billboard. This is done by considering a sub-rectangle inside a texture and generating relative UV texture coords for that sub-rectangle.

Parameters

totalTextureHeightHeight of the billboard supertexture in pixels.

cellPosition of the billboard subtexture in grid coords.
