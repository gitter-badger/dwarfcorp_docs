Detailed Description

A box has 6 faces. Each face has texture coordinates in a rectangle on a texture. Sometimes we want to transpose the texture coordinates so that X and Y are flipped. Why do we do this? So that we can flip textures around to align them without duplicating. This only necessary sometimes.
