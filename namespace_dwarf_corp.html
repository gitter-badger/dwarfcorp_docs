class  AStarPlanner
 The classic A star planner, but singled out for use with voxels. It should probably be made generic. It is called in a thread so that it can return new paths at its own leisurely pace. Plans are then returned via delegate. The A* planner uses a graph where nodes are Voxels, and the edges between nodes are Movements. A Movement could be something like "walk", "jump" or "fly". The planner finds the shortest list of Movements that go from a start Voxel to a goal Voxel.
