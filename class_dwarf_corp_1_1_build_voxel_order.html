Detailed Description

This designation specifies that a given voxel from a given BuildRoom should be built. A BuildRoom build designation is actually a colletion of these. NOTE: This is mostly a legacy class that comes from the time when dwarves would incrementally build rooms one voxel at a time. It is no longer so important.
